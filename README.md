# To Do List Web App

#### --Status: Completed

## Project Intro/Objective
The purpose of this project is to create a to do list web app that creates real time updates and automatically saves any inputs or outputs a user puts into the app.

### Technologies
* Node.js
* JavaScript
* EJS
* Express
* Body-Parser
* Lodash
* MongoDB
* MongoDB Atlas
* Mongoose
* Heroku
* HTML
* CSS
